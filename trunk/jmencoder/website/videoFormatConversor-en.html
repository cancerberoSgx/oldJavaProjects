Video format conversor


What is it?

The video format conversion application try to provide easy ways of converting any video file into an other video file of a different format. This means you can convert any video to, among others, one of the folowinginf formats: vcd, dvd, xvid, divx, mp4, flash. Also, it will let you select and paste your subtitles, leting you to choose subtitle location, font family, size, color, bg color, time delay, etc. All language and subtitle file format supported.

Common problems that this tool can help you:
  • You want to watch a movie in an old DVD player that only accept real DVD format (VIDEO_TS) or VCD format.
  • you want to watch a movie in a mp4 player like an iphone
  • you want to paste a movie subtitle and change the movie format.


This is a media player that supports:

  • A LOT of video formats, like avi, flv, mp4, rmbv, mpeg, divx, xvid, etc.
  • A lot of audio formats including mp3, mp4, wav, a4c, rm, etc
  • All known subtitle format support
  • A lot of options for defining subtitle style like font, size, color, background-color, transparency, etc
  • Advanced timing: easy subtitle delay and sound delay
